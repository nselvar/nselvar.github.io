R - Neural Network & SVM - Predict Success Of Bank Telemarketing
The shift towards decision making (DM) based on machine learning techniques techniques will change the way in which consumer markets and our societies function. Through ML, predictive analytics is being used by businesses to identify these patterns and major trends with the objective to improve the DM and influence future business outcomes. This project proposes an Neural Network (ANN)/ SVM approach to predict the success of telemarketing calls for selling bank long-term deposits. To validate the proposed model, we uses the bank marketing data of 41188 phone calls. The NN attains 90% of accuracy which outperforms other conventional classifiers and confirms that it is credible and valuable approach for telemarketing campaign managers. Deep Learning for NLP: Twitter Sentimental Analysis
Obtaining feedback from a few of the customers is easy. However, imagine that you could ask the whole world! You would know immediately the general opinion and could react accordingly. This machine learning solution examines twitter data for their tonality. The classification, whether a tweet is positive or negative, i.e. the tonality, is done by a deep neural network. The deep convolutional neural network architecture we constructed in this project,is a feedforward neural net using the TensorFlow library in Python. The neural network classifies tweet vectors in negative or positive using the Bag of Words model,ReLU activation function and softmax activation function, which will classify the tweet to either positive or negative.Click here for code and detailed report.
