Multi-agent LLM automation system built to assist in automating academic and career-related tasks.
AI Student Helper is a multi-agent automation framework utilizing LangGraph to orchestrate intelligent task execution for academic and professional assistance. It employs a Supervisor Agent powered by LLM-based intent recognition to route user requests dynamically to Specialized Agents, each leveraging NLP, OCR, web automation, and API integrations for task execution. The PDF Summarizer Agent extracts and processes text using PyMuPDF and LLM-based summarization models, while the YouTube Summarizer Agent utilizes speech-to-text conversion via Whisper AI for transcript generation. The Job Application Agent automates profile matching and form submission using Selenium and LinkedIn API, while the Homework Helper Agent provides interactive AI-driven problem-solving via Streamlit. The graph-based execution model ensures optimized task prioritization and resource-efficient processing, integrating AI-powered decision trees and modular workflow automation for seamless user experience.
